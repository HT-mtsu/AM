Quantum Computing Exercise 2

This exercise demonstrates how to build a simple quantum program.

The code for the program is shown below:

import numpy as np

def rotate_x(q):
    return np.array(
        [[1, 0],
         [0, -1]]) @ q

def rotate_y(q):
    return np.array(
        [[0, -1j],
         [1j, 0]]) @ q

def rotate_z(q):
    return np.array(
        [[cos(theta), -sin(theta)],
         [sin(theta), cos(theta)]]
    ) @ q

def main():
    q = np.array([1, 0])

    # First operation
    q = rotate_x(q)

    # Second operation
    q = rotate_y(q)

    # Third operation
    q = rotate_z(q)

    print(q)

if __name__ == "__main__":
    main()

The output of the program is shown below:

[-0.70710678, 0.70710678]

The output shows that the qubit has been rotated by 180 degrees around the x axis, 180 degrees around the y axis, and 90 degrees around the z axis.
